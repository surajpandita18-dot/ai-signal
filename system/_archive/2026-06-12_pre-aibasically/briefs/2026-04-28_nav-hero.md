## SiteNav.tsx

`'use client'` — needs scroll detection to change nav shadow.

### Props
```ts
interface SiteNavProps {
  signalNumber?: number  // optional, for display
}
```

### Behavior
- Sticky at top (`position: sticky; top: 0; z-index: 60`)
- On scroll > 20px: add subtle shadow `0 1px 16px rgba(0,0,0,0.06)`
- Background: `rgba(250, 250, 247, 0.88)` with `backdrop-filter: blur(12px)`
- Bottom border: `1px solid var(--border)`

### Layout
Left side — wordmark:
```tsx
<div> // flex, items-center, gap 10px
  <div> // 28×28px, bg var(--text), rounded-lg, flex center, color white, font-mono font-bold text-sm
    S
    // ::after dot: position absolute top-1 right-1, 7×7px circle, bg var(--warm), box-shadow 0 0 0 2px var(--text), animation livePulse 1.8s infinite
  </div>
  <span style={{ fontSize: 15, fontWeight: 700, letterSpacing: '-0.02em' }}>AI Signal</span>
</div>
```

Right side — nav links:
```tsx
<nav> // flex, gap 28px, items-center, font-size 14px, color var(--text-mute), font-weight 500
  <Link href="/archive">Archive</Link>
  <Link href="/about">About</Link>
  <a // subscribe CTA
    style={{ 
      color: 'var(--text)', fontWeight: 600, fontSize: 14,
      borderBottom: '1px solid var(--text)', letterSpacing: '-0.005em',
      transition: 'color 0.2s, border-color 0.2s'
    }}
    // on hover: color var(--signal), borderBottomColor var(--signal)
  >Subscribe →</a>
</nav>
```

The live pip on the S icon needs `animation: livePulse 1.8s ease-in-out infinite` — use `style={{ animation: 'livePulse 1.8s ease-in-out infinite' }}` (keyframe is defined in globals.css).

## HeroZone.tsx

Server component (no `'use client'`).

### Props
```ts
interface HeroZoneProps {
  date?: string       // e.g. "28 April 2026", defaults to today formatted
  tagline?: string    // defaults to "One signal, daily at 06:14 IST. For people who ship."
}
```

### Structure
```
<section className="hero-zone"> // custom CSS class, see inline styles below
  <div className="hero-banner"> // max-w 1200px, mx-auto, px-8, py-[88px] pb-14, text-center, relative z-1
    
    // Eyebrow pill
    <div style={{ display:'inline-flex', alignItems:'center', gap:10, margin:'0 auto 56px',
      padding:'8px 18px', background:'var(--bg-card)', border:'1px solid var(--border)',
      borderRadius: 999, boxShadow:'0 1px 2px rgba(0,0,0,0.03)' }}>
      <span style={{ width:6, height:6, background:'var(--warm)', borderRadius:'50%',
        boxShadow:'0 0 0 3px var(--warm-soft)', animation:'livePulse 1.6s ease-in-out infinite',
        display:'inline-block' }} />
      <span style={{ fontFamily:'var(--ff-mono)', fontSize:11, fontWeight:600,
        letterSpacing:'0.22em', textTransform:'uppercase', color:'var(--text-soft)' }}>
        {date} · Today's signal
      </span>
    </div>

    // Big headline
    <h1 style={{ fontFamily:'var(--ff-display)', fontSize:'clamp(54px, 8.4vw, 116px)',
      lineHeight:0.96, fontWeight:400, letterSpacing:'-0.035em', color:'var(--text)',
      margin:'0 auto', maxWidth:1000 }}>
      Today's one AI story{' '}
      <em style={{ fontStyle:'italic', color:'var(--signal)', display:'inline-block',
        transition:'all 0.5s cubic-bezier(0.2,0.8,0.2,1)' }}>that matters</em>
    </h1>

    // Sub
    <p style={{ fontSize:19, lineHeight:1.55, color:'var(--text-mute)',
      maxWidth:540, margin:'32px auto 0', letterSpacing:'-0.005em' }}>
      One signal, daily at <strong style={{ color:'var(--text)', fontWeight:600 }}>06:14 IST</strong>. For people who ship.
    </p>
  </div>
</section>
```

The `.hero-zone` section needs a background + overflow hidden — use an inline `<style>` tag OR a `style` prop:
```tsx
<section style={{
  background: `radial-gradient(ellipse 80% 50% at 30% 20%, rgba(43,91,255,0.06), transparent 65%),
    radial-gradient(ellipse 60% 40% at 75% 80%, rgba(255,107,53,0.04), transparent 70%),
    linear-gradient(180deg, #F6F7FB 0%, #FAFAF7 100%)`,
  position: 'relative',
  overflow: 'hidden',
  borderBottom: '1px solid var(--border)',
}}>
```

### Animation on headline hover
The `<em>` inside h1 should have a hover handler (this requires `'use client'` — if you use it, move the hover effect to a CSS class instead):

Add to globals.css approach: no, don't touch globals.css. 

Instead add a local `<style>` tag inside HeroZone:
```tsx
<style>{`
  .hero-headline:hover .hero-ital {
    text-shadow: 0 8px 30px rgba(43,91,255,0.35);
    transform: translateY(-3px) rotate(-1.5deg);
  }
`}</style>
```
And assign `className="hero-headline"` to `h1`, `className="hero-ital"` to `em`. This keeps it a server component.

### Scroll reveal
The outer section should have `className="reveal"` so IntersectionObserver in the main page picks it up.
