## Key Lenny insights already validated for AI Signal

From synthetic testing (2026-04-21):
  - Developer tools get first users through individual advocates, not orgs
  - "Time-to-value < 60 seconds" is the single biggest activation lever
  - Day 7 retention proxy: user has opened tool in 3 of last 7 days
  - Wrong user trap: indie builders (high curiosity, low decision budget)
  - Right user signal: asks "what should I do?" not "what happened?"

From GitHub Copilot story (Ryan J. Salva episode):
  - Technical credibility before social proof (ICP judges the signal quality)
  - Trust-building through accuracy, not through features
  - Distribution through individual contributors first, then team licenses
